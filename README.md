# joachimkliemann.de

This is the source behind joachimkliemann.de, my personal site and blog. It's built with [Eleventy](https://www.11ty.dev/), starting from the `eleventy-base-blog` template and grown from there.

## Repositories

- Eleventy: https://github.com/11ty/eleventy
- joachimkliemann.de: https://github.com/joxx/joachimkliemann.de

## Getting Started

- Want a more generic and detailed getting started guide? https://www.11ty.dev/docs/getting-started/

1. Make a directory and navigate to it:

```bash
mkdir my-blog-name
cd my-blog-name
```

2. Clone this repository:

```bash
git clone https://github.com/joxx/joachimkliemann.de.git .
```

Optional: review eleventy.config.js and \_data/metadata.js to configure site options and metadata.

3. Install dependencies:

```bash
npm install
```

4. Run Eleventy

Generate a production-ready build in the \_site folder:

```bash
npm run build
```

Or start a local development server:

```bash
npm start
```

Or run debug mode to inspect Eleventy internals:

```bash
npm run debug
```
